Clinical Trials XML Validator (XSD)

How to use this validator
  1. Select your file: Click the "Choose File" button below to upload your monthly clinical trials XML export.
  2. Run Validation: Click "Run Validation" to process the data against the current ICTRP XML Schema Definition (XSD).
  3. Review Results: Valid trials will appear in the "Success" tab, while structural errors or missing data will be detailed in the "Invalid" tab.
  4. Download Reports: Use the download buttons provided after validation to export valid XML subsets or error logs (CSV/ZIP).
